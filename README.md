# WorldwideMonkeys
A website where you can create your own digital monkeys, and see other people's monkeys too!

For local setup

Clone repo: git clone https://github.com/CrunchyToothpicks100/WorldwideMonkeys.git

Run API: dotnet run --project UserDataappCore.Api.csproj

Run HTTP server: npx http-server .\Front-end\ -p 7264

Connect to localhost:7264/index.html on your browser
